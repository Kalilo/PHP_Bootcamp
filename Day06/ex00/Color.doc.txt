<- Color -----------------------------------------------------------------------

This class is used for easy colour manipulation.
The class breaks down a colour into it's rgb values,
	which can be separatly handled.

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

function add($colour);
Takes a given colour and adds it to the saved colour.

function sub($colour);
Takes a given colour and subtracts it from the saved colour.

function mult($value);
Takes a given value and multiplies the saved colour by that value.
